## 🚀 Featured Projects

### 🦀 [For Legal Reasons This Is Game](https://github.com/for-legal-reasons-this-is-game/For-legal-reasons-this-is-game) · `Rust` `Docker`
An API-first, real-time **simulated trading platform** for paper-trading stocks, coins, and options. Built as a modular, horizontally scalable system with a **Rust** backend, a dedicated trading engine that recalculates prices from platform activity, and a custom relational data layer — all containerized with Docker. *Team project (ft_transcendence).*

### 🌐 [Webserv — HTTP/1.1 Server](https://github.com/ChomaMichal/webserver) · `C++98`
A from-scratch **HTTP/1.1 web server** with a non-blocking, event-driven core built on `poll()`. Parses an NGINX-inspired config, handles **GET / POST / DELETE**, static file serving, autoindex, redirects, file uploads, custom error pages, and **CGI** execution. *Team project.*

### 🐚 [Minishell — Unix Shell](https://github.com/mehrasmeydani/minishell) · `C`
A lightweight **bash-like shell** in C: tokenizing and parsing, pipes, input/output redirection, heredocs, environment-variable expansion, quote handling, signal management, and built-ins (`cd`, `echo`, `export`, `unset`, `env`, `pwd`, `exit`). Manages processes with `fork` / `execve` and careful memory cleanup.

### 🍝 [Philosophers — Concurrency](https://github.com/mehrasmeydani/philo) · `C`
The classic **dining philosophers** problem, solved with **threads and mutexes** in C. A focused study in concurrency, race-condition prevention, deadlock avoidance, and precise timing.

### 🎮 [2048 AI](https://github.com/mehrasmeydani/2048_AI) · `C++` `Python`
A hybrid **C++ / Python** project building toward a **reinforcement-learning** 2048 agent — a fast C++ game backend paired with a Python RL environment, working from baseline heuristics toward trainable DQN/PPO policies.
